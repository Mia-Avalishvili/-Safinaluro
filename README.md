# 🎸 Safinaluro – Guitar Store Website

ეს არის ფინალური HTML–CSS პროექტი, რომელიც წარმოადგენს გიტარების ონლაინ მაღაზიის (catalog-style) ვებსაიტს. პროექტის მიზანია თანამედროვე, სტრუქტურირებული და რესპონსიული ვებსაიტის შექმნა, სადაც მომხმარებელს შეუძლია პროდუქციის დათვალიერება და საკონტაქტო ინფორმაციის მიღება.

საიტი აგებულია სუფთა HTML5-ისა და CSS3-ის გამოყენებით და ოპტიმიზებულია სხვადასხვა ზომის ეკრანებისთვის.


# 📄 პროექტის გვერდები და ფუნქციონალი
🏠 Home (index.html)

მთავარი გვერდი, სადაც წარმოდგენილია:

- Header ნავიგაციით (Home, Catalog, Contact Us)

- ლოგო და იკონები (search, user, cart)

- ბანერის სექცია (“New Collection”)

Most Popular პროდუქციის სექცია:

- პროდუქტები წარმოდგენილია ბარათების (cards) სახით

- განლაგებისთვის გამოყენებულია CSS Grid

   
Footer ბრენდების, ინფორმაციისა და საკონტაქტო დეტალებით
